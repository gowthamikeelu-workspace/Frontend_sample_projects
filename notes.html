Two ways to center an inline element (like an (img)) using a (div) container: Wrap the image in a (div) and apply text-align: center to the (div). This works because the image is inline by default.

- Setting display: block makes the image behave like a block element (like a (div)), which allows it to be centered using margins. - margin: auto; This tells the browser: “Give equal left and right margins so the image sits in the center.”
